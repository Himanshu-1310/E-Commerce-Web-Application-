# 🛒 E-Commerce Web Application

## 🚀 Overview

This is a full-stack **E-Commerce Web Application** built using the MERN stack. It provides a complete online shopping experience with separate **User** and **Admin** functionalities, including product browsing, cart management, and order processing.

---

## 🛠 Tech Stack

**Frontend**

* React.js
* Redux Toolkit
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB (Mongoose)

---

## ✨ Features

### 👤 User Features

* User authentication (Sign up / Login)
* Browse products with detailed view
* Add to cart and manage cart items
* Checkout functionality
* Product search
* Product reviews

---

### 🛠 Admin Features

* Admin authentication
* Admin dashboard
* Add, update, and delete products
* Manage product listings

---

### ⚙️ Backend Features

* RESTful APIs for products, users, and orders
* Secure authentication and authorization
* MongoDB for efficient data storage
